Firestore Highlight Fields: Settings

Enter rules to highlight specific fields in the Firestore Emulator or Production Console.
Format: Project | Database | Collection list in order | Fields
(Use // to add comments)

Components:
  • Project: Your Firebase project ID (e.g., vj-project). In emulator, often default.
  • Database: Usually default or -default-.
  • Collection list in order: Comma-separated names of collections in the path (e.g., organizations, groups).
  • Fields: Comma-separated field names to highlight (e.g., name, status).
Examples:
vj-project | -default- | organizations, groups | name, status, billing.state
-ignored in emulator- | default | questions | learning.objective, examTags (Emulator)
