Highlight Settings

Enter rules to highlight specific fields in the Firestore Emulator or Production Console.
Format: Project | Database | Collection list in order | Fields

Components:
  • Project: Your Firebase project ID (e.g., utilery-project). In emulator, this field is ignored.
  • Database: Usually default in emulator and -default- in prod.
  • Collection list in order: Comma-separated names of collections in the path in order (e.g., organizations, groups).
  • Fields: Comma-separated field names to highlight (e.g., name, status, address.state).
Examples:
utilery-project | -default- | organizations, groups | name, description: to highlight name and description fields in /utilery-project/firestore/databases/-default-/organizations/*/groups/*
